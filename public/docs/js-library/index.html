Maesh.js Documentation
Maesh.js is a browser-side JavaScript library that lets you integrate Maesh into your platform. This documentation will help you get started with integrating Maesh either using the Vanilla JS CDN or the NPM library.

The libraries below are for evaluation purposes. Please contact us at hello@maesh.io or contact through our chat widget at https://maesh.io for the production build
