Prestashop Module

Learn how to integrate the Maesh module into your Prestashop store to start accepting bank app payments. No setup fees, no monthly fees. Start accepting PayNow payments in minutes with Maesh.

1. Prerequisites

  • Running Prestashop store
  • Access to the admin site of your Prestashop store
  • A valid UEN tied to your business that is registered with PayNow corporate

2. Installing the Module

  1. Download the Maesh Prestashop module here .
  2. Login to your admin dashboard.
  3. Navigate to your Module Manager and click on Upload a module.
  4. Upload the Maesh Prestashop module to your site. No need to unzip - Prestashop can handle the plugin in the same format as you've downloaded it.

3. Configuring the Plugin

  1. Create a maesh.io account. You can sign up for it here . Or if you already have an account, you can log in here .
  2. Generate a refresh token under the settings tab of the Maesh dashboard.
  3. Back on your Prestashop module manager, scroll to the Maesh module and click Configure.

  4. Paste the refresh token generated by the Maesh dashboard into the refresh token field. Don't forget to click save!

  5. You're all set! If you encounter any problems, contact us at hello@maesh.io

4. Notes

  • Feel free to message us at hello@maesh.io for any questions, concerns or suggestions - we're always looking for ways to serve you better!
  • For transactions valued greater than S$1000, customers must add Maesh as a payee for the transaction to succeed. You may disable this option in the Maesh module configuration page.
