This documentation will help you integrate Maesh as a component using the vanilla JS CDN version of Maesh.js.

Get started

Go to Settings Tab in Maesh Dashboard

To use this library, you need a Maesh API key. Create an account with Maesh to get your own API key. If you already have an account, login to find your API key.

Once you're in your Maesh Dashboard after sign-up/sign-in, go to Settings tab in the dashboard and find the API key section to generate or view your API key.

Include Maesh.js

Include the Maesh.js library on your checkout page or payment page by adding the script tag to the ​head​ tag of your HTML file. 

Initialise Maesh


                    const maesh = Maesh();

Initialise the Maesh function by assigning it to a variable. This is the entry point into Maesh.js library for creating a Maesh component

Create Maesh Component


maesh.create({ 
      api_key : 'YOUR_API_KEY', 
      dom_element_id : ​'maesh'​, 
      currency: ​'SGD'​, 
      amount: ​21700​, 
      gotoUrl: ​'https://merchant.com/success'​, 
      referenceCode: ​'ABCDEFGH' 
    })

Create an instance of Maesh in your website by calling the method ​create, which expects an object with below parameters as input. 

OBJECT PARAMETERS

Param Type Description
api_key String Your API key. This key identifies you to Maesh.
Eg. sk39Josr78Ko23Mjftlp(Dummy)
dom_element_id String This is the id of an empty HTML div element in your website, you want Maesh component to load in.
amount Integer Only ​positive integers​ allowed. Amount is in a currency's​ ​smallest unit. Maximum amount allowed is S$200,000 due to FAST transaction upper limit
Eg. S$ 8.88 should be supplied as 888
currency String The currency you want your payment to be received in. Maximum characters allowed is 3
Eg. Singapore Dollar is SGD
gotoUrl String The ​goto URL which will be invoked by Maesh after a payment status change has been detected i.e. maybe your order confirmation page.
Eg. https://merchant.com/success
referenceCode String The merchant’s reference code for the order. Should be unique. Maximum characters allowed is 10.
Eg. ABCDEFGH

Maesh Component

This instance of Maesh would generate and display a QR code in your website page for the user to scan the QR and pay using any of the bank applications supporting  PayNow. 

This is how the Maesh component is going to look like once it is successfully integrated. Once this payment is successfully completed, Maesh redirects the consumer to the URL (gotoUrl) that you provided. 

Handling Callback in the redirected page

In the redirected page, include a Maesh callback function to check the status of the  transaction, this is to ensure that the payment has been completed correctly. This  function will return the status of the transaction helping you with doing some additional tasks.  

Include Maesh.js

In the redirected page, include the Maesh.js library on your checkout page or payment page by adding the script tag to the ​head​ tag of your HTML file.
